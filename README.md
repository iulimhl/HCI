# Intelliden

You are a specialized engineer passionate about home automation.  
So passionate that you build a smart home ecosystem for your current living space, where you control it via your smartphone/tablet, smartwatch, and AR glasses.

From adding new devices, viewing information and statistics, setting automations and gestures, to interacting co-op with others (family or roommates) and much more.

---

## Design For

- Tablet / Mobile Phone
- AR Headset
- Smartwatch

---

## Run Locally

Start a local server:

```bash
python -m http.server 8080

```

Then open: http://localhost:8080/Intelliden.html
